# Cryptocurrency APP 

> Cryptocurrency Vue.js/Nuxt application that displays information about the latest cryptocurrency prices.

## Requirements
- Node >= 16.x

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev
```
Access: http://localhost:3000
